## ข้อ 9. สำรวจนักเรียน 40 คน (Venn Diagram)
**โจทย์:**
- นักเรียนทั้งหมด = 40 คน
- เล่นฟุตบอล = 18 คน
- เล่นบาสเก็ตบอล = 15 คน
**แผนภาพเวนน์-ออยเลอร์ (Venn-Euler Diagram):**

```text
       กรอบสี่เหลี่ยม: นักเรียนทั้งหมด (U) = 40 คน
  __________________________________________________
 |                                                  |
 |    เล่นฟุตบอล (18)             เล่นบาสเก็ตบอล (15)    |
 |       _______                  _______           |
 |     /         \              /         \         |
 |    /           \            /           \        |
 |   |             |          |             |       |
 |   |      13     |    5     |     10      |       |
 |   |   (บอลอย่าง   | (เล่นทั้งคู่) |  (บาสอย่าง   |       |
 |    \   เดียว)    /            \   เดียว)    /        |
 |     \_________/              \_________/         |
 |                                                  |
 |           ไม่เล่นกีฬาอะไรเลย (ด้านนอกวงกลม) = 12 คน      |
 |__________________________________________________|
```

**วิธีคิดจากแผนภาพ:**
1. วาดกรอบสี่เหลี่ยมแทนนักเรียนทั้งหมด **40 คน** และมีคนอยู่ด้านนอกวงกลม **12 คน** (ไม่เล่นกีฬา)
2. แสดงว่าคนที่อยู่ในวงกลมทั้งสองวงรวมกันต้องมี $40 - 12 = $ **$28$ คน**
3. วงกลมฟุตบอลมี **18 คน** วงกลมบาสมี **15 คน** (บวกกันได้ 33 คน)
4. ส่วนที่ทับซ้อนกันตรงกลาง (เล่นทั้งคู่) คือจำนวนที่เกินมา: $33 - 28 = $ **$5$ คน**
5. โจทย์ถามหาคนที่ **"เล่นบาสเก็ตบอล แต่ไม่เล่นฟุตบอล"** (เสี้ยวขวาของวงกลมบาส)
   ก็คือเอาวงกลมบาสทั้งหมด (15) หักตรงกลางที่ทับซ้อนกัน (5) = **10 คน**
6. **หาความน่าจะเป็น:** นำจำนวนที่หาได้มาหารด้วยจำนวนคนทั้งหมด 
   $\frac{10}{40} = \frac{1}{4} = 0.25$

**คำตอบข้อ 9:** ความน่าจะเป็นคือ **$\frac{1}{4}$** หรือ **25%**

## ข้อ 10. ตะกร้าผลไม้ (Probability)
**โจทย์:** 
มีส้ม มังคุด มะม่วง รวม 10 ลูก
- ส้ม = 2 เท่าของมังคุด
- มะม่วง = 1 ลูก
หยิบ 3 ลูกพร้อมกัน หาความน่าจะเป็นที่จะได้ผลไม้ชนิดละ 1 ลูก

**วิธีคิด:**
1. หากจำนวนผลไม้: 
   ให้ มังคุด = $M$, ส้ม = $2M$
   ส้ม + มังคุด + มะม่วง(1) = 10
   $2M + M + 1 = 10$
   $3M = 9 \implies M = 3$
   สรุป: มี **มังคุด 3 ลูก, ส้ม 6 ลูก, มะม่วง 1 ลูก**
2. หาจำนวนวิธีหยิบผลไม้ 3 ลูกจาก 10 ลูก (Sample Space)
   $n(S) = \binom{10}{3} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120$ วิธี
3. หาจำนวนวิธีที่จะหยิบได้ชนิดละ 1 ลูกพอดี (Event)
   $n(E) = \binom{6}{1} \times \binom{3}{1} \times \binom{1}{1} = 6 \times 3 \times 1 = 18$ วิธี
4. ความน่าจะเป็น $P(E) = \frac{n(E)}{n(S)}$
   $P(E) = \frac{18}{120} = \frac{3}{20} = 0.15$

**คำตอบข้อ 10:** ความน่าจะเป็นคือ **$\frac{3}{20}$** หรือ **15%**

## ข้อ 11. บวกเลขด้วยวิธีใดก็ได้ ใช้เฉพาะเลข 8 ให้ได้ 1,000
**วิธีคิด:**
ใช้เฉพาะเลข 8 นำมาบวกกันให้ได้ 1,000 พอดี 
การจัดเรียงคือการนำเลข 8 มาต่อกันเป็นหลักร้อย หลักสิบ และหลักหน่วย ดังนี้:

  `888 + 88 + 8 + 8 + 8 = 1000`

**คำตอบข้อ 11:** **888 + 88 + 8 + 8 + 8 = 1,000**

## ข้อ 12. ทอดลูกเต๋า 2 ลูก 1 ครั้ง
**โจทย์:** หาความน่าจะเป็นที่...
1. ผลรวมแต้มเป็น 10
2. ผลต่างแต้มเป็น 2

จำนวนเหตุการณ์ทั้งหมดของการทอยลูกเต๋า 2 ลูก $n(S) = 6 \times 6 = 36$ วิธี

**1. ผลรวมของแต้มเป็น 10**
- เหตุการณ์ที่เกิดขึ้นได้: (4,6), (5,5), (6,4) มีทั้งหมด 3 เหตุการณ์
- ความน่าจะเป็น = $\frac{3}{36} = \frac{1}{12}$
**ตอบ:** **$\frac{1}{12}$**

**2. ผลต่างของแต้มเป็น 2**
- เหตุการณ์ที่เกิดขึ้นได้: (1,3), (3,1), (2,4), (4,2), (3,5), (5,3), (4,6), (6,4) มีทั้งหมด 8 เหตุการณ์
- ความน่าจะเป็น = $\frac{8}{36} = \frac{2}{9}$
**ตอบ:** **$\frac{2}{9}$**
